QMkit
============

Gamess interface for python

"API以外はできるだけmolを第一引数にとる関数として実装すること"

Features
=========

- Structure optimization

- UV-VIS

- IR


Setting up
==========

Python setup
------------

.. code-block :: bash
    conda create -n qm python=3.7
    conda activate qm
    conda install psi4 psi4-rt python=3.7 -c psi4
    conda install -c conda-forge rdkit
